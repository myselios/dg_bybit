## 🎯 개선 전략 (3 Phase)

| 파라미터 | 현재 | 변경 | 효과 |
|----------|------|------|------|
| **Grid spacing** | 0.2 ATR | **0.5 ATR** | 진입 60% 감소 |
| **TP 목표** | 1.0 ATR | **2.0 ATR** | 수익 2배 |
| **MA slope 필터** | 없음 | **0.3%** | 횡보 진입 차단 |

**예상 결과:**
- 거래 횟수: 15회 → **6회** (60% ↓)
- 수수료: $3.47 → **$1.38** (60% ↓)
- 거래당 수익: $0.24 → **$0.50** (2x)
- 순수익: $0.17 → **$1.62** (9.5x!) 🚀

### Phase 2: 품질 향상 (1주 내)

**목표:** 승률 개선 + Maker 주문

**1️⃣ Maker 주문 우선**
- 현재: 대부분 Taker (0.055%)
- 변경: Limit 주문 우선 (0.02%)
- 효과: 수수료 60% 절감

**2️⃣ 시장 필터 강화**
```yaml
entry_filters:
  - MA slope > 0.3% (상승 추세)
  - ATR percentile > 40 (변동성 충분)
  - Funding rate < 0.02% (펀딩 부담 낮음)
  - Price > MA20 (지지선 위)
```

**3️⃣ 손절 최적화**
- 현재: 0.5 ATR (너무 타이트)
- 변경: 0.7 ATR (노이즈 회피)
- 효과: 손절 빈도 30% 감소

**예상 결과:**
- 승률: 50% → **65%**
- Maker 비중: 0% → **70%**
- 수수료: $1.38 → **$0.60**
- 순수익: $1.62 → **$2.50**

### Phase 3: 고도화 (1개월)

**목표:** 시장 적응력 향상

**1️⃣ 동적 파라미터**
```python
if market_regime == "trending":
    tp_multiplier = 3.0  # 큰 수익
    grid_spacing = 0.7    # 적게 진입
elif market_regime == "ranging":
    tp_multiplier = 1.5  # 작은 수익
    grid_spacing = 0.3    # 자주 진입
```

**2️⃣ 시간대 필터**
- 고변동성: 한국 21-01시 (적극 진입)
- 저변동성: 한국 06-12시 (진입 자제)

**3️⃣ 포지션 크기 조절**
```python
if atr_percentile > 70:  # 고변동성
    position_size *= 0.7  # 작게
else:
    position_size *= 1.0  # 정상
```

**예상 결과:**
- 거래당 수익: $0.50 → **$0.80**
- 승률: 65% → **70%**
- 월 수익: 3% → **10-15%**
